
# Cómo abrir una cuenta Meter Para taxista: Guía paso a Paso

## ¿Cómo crear una cuenta Meter for Taxis?
Para crear una cuenta Meter, estos son los pasos que debes seguir:

1. Descarga la aplicación de Meter for Taxis en la Play Store o la App Store.
- [Apple App Store](https://apps.apple.com/us/app/meter-for-taxis/id931234419)
- [Google Play Store](https://play.google.com/store/apps/details?id=osacky.ridemeter)

2. Abre la aplicación y oprime el botón "Ajustes".
<img height="150" src="/screenshots/IMG_8965.png" width="250"/>

3. Selecciona "iniciar sesión", ahora escribe tu correo electrónico y selecciona "OK".
<img height="150" src="/screenshots/IMG_8966.png" width="250"/>

4. Recibirás un correo electrónico con un código para iniciar sesión, - digita el código y selecciona "OK".
<img height="150" src="/screenshots/IMG_8967.png" width="250"/>

5. Personaliza tu perfil: Nombres o nombre comercial.
<img height="150" src="/screenshots/IMG_8968.png" width="250"/>

¡Listo! Una vez completado los pasos, ya estás registrado y puedes revisar todas las ventajas de usar nuestro software Taxímetro.
