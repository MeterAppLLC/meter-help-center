# ¿Cómo seleccionar  el tipo de  moneda en  iOS? ?

Sigue estos pasos:

* Abre la aplicación y navega hasta la pantalla principal.
* Busca la opción etiquetada como "Viaje" y tócala.
* Selecciona " administrar  tarifas personalizadas".
* Aquí, verás la opción de seleccionar una tarifa existente o crear una nueva.
* Para crear una nueva tarifa, toca el ícono de suma ubicado en la esquina superior derecha de la pantalla.
* Después de seleccionar una tarifa existente o crear una nueva, desliza hacia abajo en la pantalla.
* Busca y toca la opción etiquetada como "Moneda".
* Esto te permitirá especificar la moneda en la que deseas definir la tarifa.
