# Idiomas Disponibles para la Aplicación Meter

La aplicación Meter está disponible en varios idiomas para atender a una audiencia global. A continuación se muestran los idiomas compatibles para las plataformas Android y iOS:

## Idiomas para Android

- Inglés
- Amárico
- Árabe
- Chino (Simplificado)
- Chino (Tradicional)
- Croata
- Holandés
- Francés
- Alemán
- Hindi
- Húngaro
- Indonesio
- Italiano
- Japonés
- Coreano
- Portugués
- Ruso
- Español
- Tailandés
- Turco
- Ucraniano
- Vietnamita

## Idiomas para iOS

- Inglés
- Español
- Árabe
- Chino (Simplificado)
- Chino (Tradicional)
- Holandés
- Francés
- Alemán
- Japonés
- Coreano
- Portugués (Portugal)
- Ruso
- Turco

> **Nota:** La disponibilidad de ciertos idiomas puede depender de la configuración de idioma de tu dispositivo y la disponibilidad regional.
