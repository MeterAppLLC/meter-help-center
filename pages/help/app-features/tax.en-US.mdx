# How to Add Taxes?

## Android Users:

Follow these steps:

* Select "trip".
* Tap "Manage Custom fares".
* You can modify an existing rate to add taxes by tapping on it.
* To add taxes to a new rate, tap the "+" symbol in the upper right corner.

# iOS Users:

Follow these steps:

1. Tap "Settings".
2. Press "Tax rate".
3. In the window that opens, add the tax percentage you want to charge.
4. To save, select "Done".
