# How to Add Taxes?

## Android Users:

Follow these steps:

* Select "trip".
* Tap "Manage Custom fares".
* You can modify an existing rate to add taxes by tapping on it.
* To add taxes to a new rate, tap the "+" symbol in the upper right corner.

# iOS Users:

Follow these steps:

* Tap "Settings".
*  Press "Tax rate".
* In the window that opens, add the tax percentage you want to charge.
* To save, select "Done".
