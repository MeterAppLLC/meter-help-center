# ¿Cómo agregar impuestos?

# Usuarios de Android:

Sigue estos pasos:

* Selecciona "viaje".
* Toca "Administrar tarifas personalizadas".
* Puedes modificar una tarifa existente para agregar impuestos presionando sobre ella.
* Para agregar impuestos a una tarifa nueva, toca el símbolo "+" en la parte superior derecha.

# Usuarios de iOS:

Sigue estos pasos:

1. Toca "Ajustes".
2. Presiona "Porcentaje de impuesto".
3. En la ventana que se abre, añade el porcentaje de impuesto que deseas cobrar.
4. Para guardar, selecciona "Hecho".
