# Activar la navegación GPS
Sigue estos pasos:

* Abre la aplicación y presiona el icono de "Soporte".
* Presiona "Configuración de navegación".
* Desliza el botón  para  "Predeterminar  al modo de navegación"

Esta disponible en suscriptores Meter Pro y miembros de una Flota- Empresa
