# Para activar la navegación GPS
 sigue estos pasos:

* Abre la aplicación y presiona el icono de ajustes.
* Selecciona "Ajustes de navegación".
* Presiona el botón verde para activar o desactivar las alertas de sonido.
* Activa la opción de navegación presionando el botón verde.

Estos pasos aplican tanto a usuarios Meter Pro como a miembros de una flota.
