# Para activar la navegación GPS
 Sigue estos pasos:

* Abre la aplicación y presiona el icono de ajustes.
* Selecciona "Ajustes de navegación".
*  Desliza  el botón verde para activar o desactivar las alertas de sonido y acitivar navegación

Esta disponible en suscriptores Meter Pro y Flota- Empresa
