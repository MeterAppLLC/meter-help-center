# Despachar un conductor

* Ahora puedes despachar conductores en tu flota
* Utiliza la aplicación de Android para despachar un conductor a una ubicación de recogida en tiempo real. El conductor recibirá una notificación y podrá navegar hasta la ubicación de recogida.

**Opciones de navegación**

* Elige tu opción de navegación favorita. Google Maps, Waze o Meter (impulsado por Google Maps). ¡Recomendamos usar Meter para navegar y obtener la mejor experiencia!
