# Cómo Invitar a un Conductor a Unirse a la Flota?

Sigue estos pasos:

* Inicia sesión en tu cuenta de flota a través de la app.
* Presiona en la opcion "Flota"
* En la parte superior encontrarás " Comparte el código de ingreso" Este es el código de unión que le debes compartir con el conductor para que ingrese a la flota.
