# ¿Cómo cambio de millas a kilómetros o viceversa?

Sigue estos pasos:

* Inicia sesión en tu cuenta Meter.
* Selecciona "Viajes".
* Selecciona "Gestionar tarifas personalizadas".
* [Selecciona este enlace si deseas aprender cómo crear tarifas personalizadas](https://help.meterapp.co/es-ES/help/pro-features/custom_fares).
* Elige una tarifa existente o crea una nueva.
* Dentro de tu tarifa, desliza hacia abajo y selecciona:
   - En Android: el botón "Metric".
   - En iOS: el botón "Use Metric Units".
* Desliza para activar el cambio de millas a kilómetros o viceversa.
