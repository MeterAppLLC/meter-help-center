# ¿Cómo configurar tu número de teléfono?

Tu número de teléfono se mostrará en las facturas y recibos.

Sigue estos pasos:

* Inicia sesión en tu cuenta de Meter a través del sitio web.
* [Haz clic aquí para iniciar sesión](https://www.meterapp.co/ar/signin).
* Una vez dentro de tu perfil de cuenta, desplázate hacia abajo en la página hasta que encuentres la opción "Número de teléfono".
* Toca la opción "Actualizar número de teléfono", agrega tu número de teléfono y luego toca "Enviar teléfono". Enviaremos un código de verificación de un solo uso a tu número móvil que deberás ingresar para verificar tu número.
* Después de ingresar el código de verificación de un solo uso, toca "Enviar teléfono".
